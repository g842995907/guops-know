

#### 概述
&emsp;&emsp;场景引擎系统用来描述网络拓扑结构和标靶节点属性，场景支持基础场景、攻防场景等多种模式。标靶有多种类型，如网络、路由或终端，每种类型在网络中的定位和功能都不一样。
