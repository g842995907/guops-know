**random模块**

```python
random.random() # 生成一个 0-1 之间的随机浮点数;

random.uniform(a, b) #生成[a,b]之间的浮点数;

random.randint(a,b) # 随机a-b数字中的一个整数，包含数字a和b

random.choice(list) # 随机选择list里面的一个元素

random.shuffle(list) # 打乱列表里元素的排列顺序

random.sample(str,4) # 选取特定数量的字符串

random.randrange(a, b, step) # 在指定的集合[a,b)中,以 step 为基数随机取一个数;
```
